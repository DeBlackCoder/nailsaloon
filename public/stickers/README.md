# Stickers

Place sticker WebP files in this directory. The following files are expected:

- `heart-eyes.webp`
- `sparkles.webp`
- `nail-polish.webp`
- `fire.webp`
- `crown.webp`
- `rainbow.webp`
- `star-struck.webp`
- `clapping.webp`
- `love-letter.webp`
- `butterfly.webp`
- `cherry-blossom.webp`
- `100.webp`

If a sticker image is missing or fails to load, it will be hidden gracefully via the `onError` handler in the sticker picker UI.
